Hotel chalé nas montanhas

É um projeto que visa criar uma plataforma online para divulgar e reservar os chalés do hotel, que ficam localizados em uma região montanhosa e cercada de verde. O site permite que os clientes conheçam os detalhes e as fotos dos chalés, os preços e as disponibilidades, e façam as suas reservas de forma rápida e segura. Esse foi o meu primeiro site como dev front-end, colocando meus conhecimentos em pratica, sendo um projeto original.
Tecnologias usadas: html, css, javascript

Link do projeto: github

Multiroom Chat

O Multiroom Chat é um aplicativo web de comunicação em tempo real, desenvolvido em Node.js. O projeto utiliza o framework web Express.js para criar rotas, gerenciar requisições e respostas, e integrar diferentes tipos de middleware. O projeto também usa as bibliotecas Websockets IO para a comunicação bidirecional entre o servidor e o cliente, e outras tecnologias como: body-parser, EJS, consign

Link do projeto: github
